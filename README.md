# layouts
### Keyboard Layouts Optimized For Development
* Numbers and symbols on the top row are swapped.
* ESC and CAPSLOCK are swapped.
* `us(workman-intl)` and modified `us` ("QWERTY") layout.

### To Use
Install the files to your home folder, using `homesick`, or manually placing them.

Run / Add to your `.xinitrc`: `xkbcomp -w 0 ~/.xkbmap $DISPLAY`

Enjoy!
